# Networked Contextual HERMES Presentation Package

## Package purpose
This folder is ready to share for presentation and review of the Networked Context-Conditional Event Inference and Coordination extension.

## Included files
- `presentation_networked_contextual_hermes.html`
  - web presentation (open in browser)
- `presentation_networked_contextual_hermes.pdf`
  - static slide deck for email, print, and offline sharing
- `presentation_networked_contextual_hermes.md`
  - source slide markdown
- `assets/networked_contextual_hermes/*.png`
  - visual assets used by the web deck

## How to use
1. For live presenting: open `presentation_networked_contextual_hermes.html` in a browser.
2. For simple distribution: share `presentation_networked_contextual_hermes.pdf`.
3. For editing/rebuilds: update `presentation_networked_contextual_hermes.md` and regenerate from the repo scripts.

## Notes
- Keep the `assets/networked_contextual_hermes` folder next to the HTML deck. The HTML references those images by relative path.
- This package contains only presentation deliverables and does not require running code to view.
